# 旅行假日 - 标准Demo

【风格】旅行假日 · 海岛日落
【画幅】1024x1792 (9:16竖版)

【Prompt】
黄昏的海岛露台,一位24岁的年轻东方女性站在无边泳池边的木质平台上,身后是被夕阳染成金橙色的大海。她穿着一条米白色的棉麻吊带长裙,裙摆被海风轻轻吹起,戴着一顶宽檐草帽,脖颈上挂着简单的贝壳项链,赤脚踩在温热的木板上。

她的妆容清透自然,带着度假的松弛——健康的裸色底妆,珊瑚色的唇,被夕阳晒得微微发红的脸颊。她一手轻轻按着草帽,身体侧向大海,回眸看向镜头,长发和裙摆一起被风扬起,笑容明媚惬意,眼里映着金色的光。

背景是无边泳池与海面连成一线,远处有几株椰子树的剪影和缓缓下沉的夕阳,天空是渐变的橙紫色。强烈的金色逆光从海平面方向照来,在她的发丝和裙边勾出温暖的轮廓光,画面通透明亮。

写实旅拍质感,真实肌肤,金色时刻暖调,浅景深,自由惬意的度假氛围。
