# 日系 - 标准Demo

【风格】日系清新 · 春日公园
【画幅】1024x1792 (9:16竖版)

【Prompt】
春日午后的城市公园，一位24岁的年轻东亚女性站在草地上，身后是盛开的樱花树。她穿着白色棉质连衣裙，裙摆自然垂落到膝盖以下，袖口宽松，领口简洁。长发自然披散在肩上，发丝被微风轻轻吹起。

她的妆容非常淡雅自然——几乎不着痕迹，只有淡淡的粉色腮红和裸粉色唇妆，皮肤通透清透。表情温柔放松，眼神看向镜头外的远方，嘴角带着浅浅的微笑。一只手轻轻提着裙摆，另一只手自然垂在身侧。

背景是柔和虚化的樱花树和绿色草地，远处有模糊的树影。春日午后的自然光从她身后照来，形成柔和的逆光，在发丝边缘勾勒出金色的轮廓光，白色裙子微微透光。空气中有细微的光斑和柔和的散景，整体氛围清新治愈。

Kodak Portra 400胶片质感，柔焦，自然光线，轻微颗粒感，暖调柔和色彩，真实皮肤纹理，浅景深。
