# 低调电影感 - 标准Demo

【风格】低调电影感 · 夜窗静帧
【画幅】1024x1792 (9:16竖版)

【Prompt】
一个昏暗的夜间室内,一位27岁的年轻东方女性坐在窗边,像一帧电影剧照。室内几乎没有开灯,唯一的光来自窗外街灯透进的一束冷调光线,斜斜地照亮她的半张脸和肩颈,其余部分沉入柔和的暗影里——但暗部依然可读,能看清她深色针织衫的纹理和背景里模糊的家具轮廓。

她微微低着头,一手支在窗台上,目光望向窗外,神情沉静里带着一丝心事,眼睛里有街灯的微光。发丝的边缘被冷光勾出一道细细的轮廓。整个人有一种克制的情绪张力,像故事里某个安静的转折瞬间。

背景是深色的墙面和虚化的窗棂,窗外是散成光斑的夜色。低饱和的电影调色,冷暖分离——冷调的窗光与暗部里一点残留的暖意,阴影厚重但有层次。

低照度电影静帧质感,可读暗部,低饱和电影调色,真实肌肤,叙事情绪感,50mm电影镜头。
