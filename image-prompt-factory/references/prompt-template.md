# 标准 Prompt 结构与写作规范

本文件定义面向 **GPT-image / DALL-E** 的通用 prompt 结构和写作方法。

> 核心理念:**先在脑中把画面看清楚,再用自然语言流畅地描述出来**,而不是逐格填模板。
> GPT-image 对段落式的自然语言、场景感和内部逻辑响应最好;它**不支持** negative prompt,也不需要 SD 式咒语或 MJ 式参数。

## 输出格式

```
【风格】[风格名称 · 子分类描述]
【画幅】[1024x1792 (9:16) / 1792x1024 (16:9) / 1024x1024 (1:1)]

【Prompt】
[自然流畅的段落式描述,通常 2-4 段]
第1段: 场景 + 人物主体
第2段: 服装 + 妆容细节
第3段: 姿态 + 场景背景
第4段: 光线 + 镜头 + 质感

[语言按风格选择:古风中文、3D CG 英文、韩系中英混合]
[长度让画面决定,150-500 字,不要硬凑形容词]
```

**要素拆解为可选项**:仅当用户明确要"拆解/清单"时,才在 prompt 之后附上人物/服装/场景/光线/镜头等分点说明。默认只输出上面的段落式 prompt。

## 段落式写作的内在逻辑

自然语言 prompt 的质量不在于要素齐全,而在于**要素之间彼此呼应**,让 GPT-image 读到一个连贯的画面而非零散标签:

- **光线 → 阴影 → 质感**: 说了"侧逆光从右后方",就该有"发丝边缘的轮廓光""面部的柔光补亮"
- **姿势 → 表情 → 眼神**: "回眸"就该配"侧转的身体""看向镜头的眼神"
- **材质 → 反光**: "缎面吊带""玻璃唇釉"要落到"湿润高光""水润反光"
- **场景 → 氛围 → 色调**: "深夜咖啡馆"自然带出"暖金吊灯""窗外雨夜光斑""慵懒氛围"

写的时候在脑中"看"这一刻正在发生什么,再把看到的顺着逻辑写下来。

## 内部自检要素(不逐条输出,用来确保画面完整)

下面 8 类是动笔前在脑中过一遍的**检查项**,不是输出模板。它们保证段落里不漏关键信息,但最终要融进自然语言,而非列成清单。

### 1. 人物设定
年龄段(明确成年,如 22-28 岁东方女性)、五官轮廓、整体气质(清纯/知性/冷艳/甜美/高贵)、表情与眼神方向。

### 2. 妆容发型
妆容风格(水光妆/哑光妆/韩式玻璃肌)、底妆质感、眼妆、腮红、唇妆质感(哑光/水光/玻璃唇)、发型发色、发饰。

### 3. 服装配饰
款式(汉服/连衣裙/针织衫)、颜色搭配、面料材质(真丝/棉质/雪纺/缎面)、设计细节(领口/袖型/刺绣)、首饰鞋包眼镜等配饰。

### 4. 身材体态
整体身形、需要突出的线条(肩颈/腰线/腿部)、具体姿势、手部动作与站/坐姿细节。描述得体、不低俗。

### 5. 场景环境
室内/户外与具体场所、背景物件、空间纵深、前景与背景的层次安排。

### 6. 光线氛围
光源(自然光/人造光/混合)、方向(顺光/侧光/逆光/侧逆光)、色温、光效(柔光/硬光/体积光/轮廓光)、明暗对比。

### 7. 镜头语言
景别(特写/半身/全身)、视角(平视/俯视/仰视)、焦段感(35mm/50mm/85mm)、景深、构图。GPT-image 用自然语言表达即可,不写 MJ 参数。

### 8. 画面质感
写实摄影/电影级/3D 渲染等质感取向、后期风格(胶片/数码清透/电影调色)、色调、特殊效果(颗粒感/柔焦/光晕/轻微过曝)。这些用描述性短句融进结尾段落,而非堆砌 `8k, masterpiece` 之类咒语。

## 语言选择指南

- **古风/国风/仙侠**: 中文为主,意境和材质细节用中文表达最到位
- **日系**: 中文为主,胶片/柔焦等技术参数可用英文(如 `Kodak Portra 400`)
- **韩系/CCD**: 中英混合,韩式术语 + CCD/闪光/颗粒等技术词用英文
- **都市/职场/居家/泳装/生活写真**: 中文为主,摄影术语按需用英文
- **3D CG/幻想**: 英文为主,保留渲染引擎/材质/光照的技术准确性

## 长度控制指南

- **简单场景** (150-250字): 单一场景、简洁元素
- **中等复杂** (250-350字): 多层次描述、丰富细节
- **高度精细** (350-500字): 古风重工、多重材质、复杂场景

不要为了凑字数堆砌形容词——让画面的复杂度决定长度。

## 不要输出

- ❌ **Negative prompt** — GPT-image 不支持,写了也无效
- ❌ **SD 式咒语** — `masterpiece, best quality, 8k` 这类堆词
- ❌ **MJ 式参数** — `--ar 9:16`、`--v 6` 等命令行参数
- ❌ **强制的 8 要素 bullet 清单** — 除非用户明确要"拆解"

## 安全检查清单(强制)

生成 prompt 前逐项确认:
- [ ] 年龄明确标注为成年(22-28岁等)
- [ ] 避免任何未成年 / 幼态相关描述
- [ ] 人体比例合理,手部解剖正确
- [ ] 不冒用真人姓名或身份
- [ ] 避免低俗、暴露、暴力、血腥内容
- [ ] 场景与风格匹配,光线描述具体可执行
