# 3D CG/角色/幻想 风格DNA

## 触发关键词
3D、CG、游戏角色、次世代、幻想、渲染、哥特、暗黑、天使、恶魔、科幻

## 核心视觉特征
- **风格**: 3D渲染、半写实、游戏角色、次世代建模
- **光线**: 电影级打光、体积光、边缘光、dramatic lighting
- **材质**: PBR材质、次表面散射、金属反射、布料物理
- **场景**: 幻想场景、暗黑哥特、天界宫殿、未来科幻
- **氛围**: 幻想、戏剧化、高对比、视觉冲击力

## 语言风格
**英文为主**，保留技术术语准确性，也可中英混合。

技术术语: 3D rendering, Unreal Engine, PBR materials, subsurface scattering, volumetric lighting, ray tracing

风格词: semi-realistic, next-gen, game character, cinematic, dramatic lighting

## GPT-image 适用提示
- 明确技术标准(Unreal Engine quality, 8K, ultra detailed)
- 详细描述材质(皮肤SSS、金属反射、布料物理)
- 光线要电影级(体积光、边缘光、冷暖对比)
- 场景要有幻想感或科技感
- 可以更戏剧化、更视觉冲击
- 长度通常 250-400字
- 英文描述更准确，中文辅助
