# 技能描述优化

## 当前描述

```
Analyzes markdown articles, automatically recommends illustration positions based on section structure, generates prompts, and creates images using codex-cli or gemini backends. Use when user asks to "illustrate article", "add images to article", "generate article images", "为文章配图", or "给文章生成图片".
```

## 优化目标

1. **更清晰的功能描述**
2. **更多触发关键词**
3. **突出自动化特点**
4. **支持中英文场景**

## 优化后的描述选项

### 选项1：功能导向
```
Automatically illustrate markdown articles with AI-generated images. Analyzes article structure, recommends optimal image positions, generates detailed prompts, and creates images using codex-cli or gemini. Supports multiple styles (infographic, diagram, hand-drawn, scene). Use for "为文章配图", "add images to article", "illustrate", "generate article illustrations", "给文章生成图片", "文章插图".
```

### 选项2：场景导向
```
Add professional illustrations to technical articles, tutorials, and blog posts. Analyzes content, recommends image count and positions, generates images in various styles (infographic/diagram/hand-drawn), and inserts them into markdown. Use when you need to "illustrate article", "add visuals", "为文章配图", "generate diagrams", "create article images", "配图".
```

### 选项3：简洁高效
```
Intelligent article illustration: analyzes markdown, recommends image positions, generates styled images (infographic/diagram/hand-drawn/scene), inserts into article. Trigger: "illustrate article", "add images", "为文章配图", "generate illustrations", "给文章生成图片", "文章配图工具".
```
