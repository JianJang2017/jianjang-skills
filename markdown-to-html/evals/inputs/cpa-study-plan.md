# 中级会计职称备考60天冲刺计划

距离考试还有两个月，时间不算宽裕。这个阶段的目标不是把三科再通读一遍，而是把《会计实务》《经济法》《财务管理》排进一个能每天坚持的节奏，优先保证高频考点能默写、会算、不丢分。

> 备考后期最怕的不是不会，而是计划排得太满、一中断就再也接不上。每天留一个能完成的最低任务，比列一张完美但坚持不了的表更重要。

## 一、目前的进度

前期已经过了一轮基础，现在的短板很清楚：

- 会计实务：长期股权投资、合并报表还不熟
- 经济法：票据法、公司法的细节记不牢
- 财务管理：公式多，计算题速度上不去

## 二、60 天节奏安排

把整个冲刺切成三段，每段有明确产出和验收方式：

| 时间 | 重点任务 | 验收方式 |
| --- | --- | --- |
| 第 1-20 天 | 三科高频考点过一遍 | 每个考点能口述要点 |
| 第 21-45 天 | 真题按章节刷 | 每周一套近年真题 |
| 第 46-60 天 | 模拟卷 + 错题回炉 | 稳定控制答题时间 |

### 每日最低任务

哪怕加班很累，这几项也要完成，保证连续性：

```
1. 听课/看书 30 分钟
2. 刷题 20 道
3. 复盘错题 3 道
4. 睡前默写当天 1 个核心考点
```

## 三、上一轮复盘

上个月的计划失败了，主要是工作日排了 2 小时任务，一加班就崩。这次把底线压到 45 分钟，先求连续，再求强度。

复盘结论：计划不是越细越好，而是越容易恢复越好。当天断了，第二天必须能快速接上，而不是从头补。
