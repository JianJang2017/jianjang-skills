# 微信公众号图片规格说明

## 封面图规格

微信公众号封面图推荐尺寸：

### 首图封面（推荐）
- **尺寸**：900×383 像素
- **比例**：2.35:1
- **用途**：文章首图、分享卡片
- **格式**：JPG（质量 85-90）或 PNG
- **文件大小**：建议 < 300KB，最大 2MB

### 次选封面
- **尺寸**：900×500 像素
- **比例**：1.8:1
- **用途**：备用封面
- **格式**：JPG（质量 85-90）或 PNG
- **文件大小**：建议 < 300KB，最大 2MB

## 文内配图规格

### 横图（推荐）
- **宽度**：900-1080 像素
- **比例**：16:9 推荐
  - 900×506 像素
  - 1080×608 像素
- **格式**：JPG 或 PNG
- **文件大小**：建议 < 500KB

### 方图
- **尺寸**：900×900 像素
- **比例**：1:1
- **格式**：JPG 或 PNG
- **文件大小**：建议 < 500KB

### 竖图（少用）
- **宽度**：900 像素
- **比例**：根据内容，但不建议过长
- **格式**：JPG 或 PNG

## 微信 API 限制

根据 `wechat_api.md` 和官方文档：

### 上传接口限制
- **封面素材**：`/cgi-bin/material/add_material?type=image`
  - 格式：JPG/PNG
  - 大小：最大 2MB
  - 返回：`thumb_media_id`

- **文内图片**：`/cgi-bin/media/uploadimg`
  - 格式：JPG/PNG
  - 大小：最大 10MB（建议 < 2MB）
  - 返回：微信 CDN URL

### 移动端显示
- 公众号文章在手机上显示时，宽度自动适配屏幕
- 建议宽度 ≥ 900px 以保证高分屏清晰度
- 但不要过大（> 1080px），否则文件体积难以控制

## 技能默认配置

本技能默认生成以下规格：

| 图片类型 | 尺寸 | 比例 | 格式 |
|---------|------|------|------|
| 封面图 | 900×383 | 2.35:1 | JPG (质量 90) |
| 意境图 | 1080×608 | 16:9 | PNG 或 JPG |
| 场景图 | 1080×608 | 16:9 | PNG 或 JPG |
| 知识图解 | 900×506 | 16:9 | PNG |

## 生成参数映射

使用 `generate-image.js` 时的 `--aspect-ratio` 参数：

```bash
# 封面图
--aspect-ratio 2.35:1    # 生成 2.35:1 比例，codex/agy 会自动选择合适分辨率
                          # 通常输出 1024×435 或类似，后续可调整为 900×383

# 配图
--aspect-ratio 16:9      # 生成 16:9 比例，通常输出 1024×576 或 1792×1024
                          # 可直接使用或调整为 1080×608

--aspect-ratio 1:1       # 方图，通常输出 1024×1024
```

**注意**：
- `codex` 和 `agy` 的实际输出尺寸由模型决定，不能精确指定像素
- 如果需要精确尺寸（如严格的 900×383），需要后处理调整
- 对于封面图，推荐在 prompt 中要求保留排字安全区，文字由 `compose-cover.js` 后期合成

## 最佳实践

### 1. 直接生成目标尺寸
- 在 prompt 中说明目标用途："for WeChat article cover, 2.35:1 aspect ratio"
- 模型会生成接近目标比例的图片

### 2. 封面文字策略
- **无字底图**：先生成干净背景（使用 `generate-image.js` 默认模式）
- **文字合成**：使用 `compose-cover.js` 叠加诗名、作者、诗句
- **短诗**（≤4行）：显示全文
- **长诗**（>4行）：显示 1-2 句代表名句

### 3. 文件大小控制
- 场景图、意境图：PNG（保留细节）或 JPG 质量 90
- 封面图：JPG 质量 85-90（平衡质量与大小）
- 知识图解：PNG（确保文字清晰）

### 4. 配图数量
- 标准配置：封面 + 2-3 张配图
- 封面（900×383）+ 意境图（1080×608）+ 知识图解（900×506）
