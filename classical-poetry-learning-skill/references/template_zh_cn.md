# 纯中文版推文模板参考

## 推文标题公式

- 标准版：`【课本精讲】《XXX》｜译文+赏析+作者生平+必考考点`
- 引流版：`必考古诗XXX：读懂诗人经历，孩子再也不会写错主旨`
- 干货版：`吃透这首XX诗，搞定小学X年级核心考点`

## 推文结构模板

### 一、诗词原文

```
[诗词完整原文]

**易错提醒**：[标注易错字、通假字、多音字]
```

### 二、诗人小档案

先用一张"档案卡片"列出关键信息，再用大白话展开生平。

```
【档案卡片】
姓　名：[姓名，字、号（若有）]
生　卒：公元[出生年]年 — 公元[去世年]年（享年[X]岁）
朝　代：[朝代（如"唐代·盛唐时期"）]
性　别：[男/女]
称　号：[核心标签，如"诗仙"；及并称，如"与杜甫并称'李杜'"]
代表作：[3—4 首代表作品]

【生平展开】
[诗人名]，[朝代][核心标签]。他大约出生在公元[出生年]年，[出生地/家世背景]。**他生活在[朝代时期特征，如"盛唐时期，国力强盛、文化繁荣，士人可以自由漫游追求理想" / "晚唐衰落期，藩镇割据、社会动荡"]，[这如何影响了他的人生/创作]。**[童年/少年关键经历]。
[成年后人生要事：仕途、漫游、重大转折，用大白话按时间顺序讲述]，于公元[去世年]年去世，享年[X]岁。
他的写作风格[核心特点+举例]，是[小学/初中]阶段考察[最多/较多]的诗人之一。
```

> 生平要事至少覆盖：出生（年份+地点）、**时代特征（1句话点明诗人所处时期的社会氛围及对其影响）**、成名/求学、仕途或重大经历、晚年与离世。年份务必核实，避免张冠李戴。时代特征要**与这个诗人直接相关**（如李白→盛唐繁荣与漫游自由，杜甫→安史之乱由盛转衰），不要写成通史教科书。

### 三、创作背景

```
这首诗大约作于公元[年份]年（[朝代年号，如"开元年间"]），当时[诗人名]约[X]岁，正值[人生阶段]。这首诗是[诗人名]在[时间/地点/人生阶段]所作。[用故事化语言描述创作情境，结合诗人境遇和时代背景]，于是写下这首[诗词特点]的[诗词类型]。
```

> 务必写清"创作时间 + 作者当时岁数"，帮孩子建立"诗人多大年纪写下这首诗"的时间感。若创作年份有争议，注明"大约"并取主流说法。

### 四、白话译文

```
[逐句对应的口语化翻译，画面感强，通俗易懂]
```

### 五、逐句赏析

```
1. [第一句/联]"[原文]"：[句子类型（写景/抒情等）]，[修辞手法]，[画面描写]，[情感分析]，[艺术效果]。

2. [第二句/联]"[原文]"：[同上结构分析]

[以此类推]
```

### 六、小学/初中必考考点

```
- **默写考点**：[必考句子]，[易错字词]，[情境默写题型]

- **赏析考点**：[主旨情感]，[修辞手法]，[炼字妙处]，[画面描写]

- **答题模板**：[高频简答题标准答题句式]

- **拓展考点**：[同作者同类诗作]，[同题材诗词对比]
```

### 七、亲子小结

```
《[诗词名]》是[诗词类型]，全诗[语言特点]、[画面特点]，核心考点是[考点1]+[考点2]+[考点3]，是[年级]必考[基础/重点]篇目，孩子必须[学习要求]。
```

### 八、扩展模块（可选，按诗词需要插入 0-2 个）

> 不是每首诗都需要扩展。只有当诗词涉及下列情况时才插入相应模块，否则跳过本节。

#### 8.1 文化小贴士

**适用场景**：诗中涉及重要典故、历史事件、传统习俗、器物制度，需要补充背景知识才能理解。

**示例场景**：
- 《赤壁》→ 补充赤壁之战、周瑜背景
- 《元日》→ 解释春节习俗（爆竹、桃符）
- 《春望》→ 补充安史之乱历史背景

**写法**：
```
#### 文化小贴士：[典故/事件/习俗名称]

[用通俗语言解释背景知识，2-3 句话即可，与诗词内容紧密相关，不展开无关历史]
```

#### 8.2 地理/物候知识

**适用场景**：诗中涉及特殊地理位置、名胜古迹、季节物候，补充后能增强理解。

**示例场景**：
- 《望庐山瀑布》→ 庐山在哪、为何有瀑布
- 《黄鹤楼送孟浩然之广陵》→ 黄鹤楼与长江航运
- 《山行》→ "霜叶红于二月花" 的物候现象

**写法**：
```
#### 地理/物候小知识：[地名/物候现象]

[解释地理位置/特色，或季节物候特征，2-3 句话，帮孩子建立空间/时间感]
```

#### 8.3 延伸阅读

**适用场景**：有经典的同题材对比作品，列出来能帮孩子建立"诗词家族"概念。

**示例场景**：
- 《静夜思》→ 对比王安石《泊船瓜洲》（同为思乡）
- 《望庐山瀑布》→ 对比苏轼《题西林壁》（同写庐山）
- 《悯农》→ 对比白居易《观刈麦》（同写农民辛苦）

**写法**：
```
#### 延伸阅读：同题材诗词

**[诗词名]** · [作者]
[诗词原文，4句以内]

**共同点**：[都是写XX主题/都用了XX手法]
**不同点**：[本诗侧重XX，对比诗侧重XX]
```

#### 8.4 亲子讨论

**适用场景**：诗词主题适合启发思考，或能联系生活实际，值得亲子共同讨论。

**示例场景**：
- 《悯农》→ "我们可以怎样珍惜粮食？"
- 《咏鹅》→ "观察身边的小动物，它们有什么特点？"
- 《登鹳雀楼》→ "生活中遇到困难时，你怎样让自己'更上一层楼'？"

**写法**：
```
#### 亲子讨论时间

💡 **[问题1]**：[开放式问题，引导孩子联系生活]
💡 **[问题2]**：[另一个角度的启发性问题，可选]
```

---

**使用原则**：
- 简单直白的诗（如《静夜思》《春晓》）→ 不加扩展，避免凑字数
- 涉及典故/历史的诗（如《赤壁》《春望》）→ 加"文化小贴士"
- 涉及名胜的诗（如《望庐山瀑布》）→ 加"地理知识"
- 同题材有经典对比的诗 → 加"延伸阅读"
- 主题适合讨论的诗（如《悯农》） → 加"亲子讨论"
- 最多插入 2 个扩展模块，不要贪多

### 九、文末固定话术

```
学好古诗词，不靠死记硬背，靠理解、靠积累、靠抓准考点。

本专栏持续同步课本必背古诗，精讲作者生平、诗词赏析、应试干货，帮孩子轻松吃透课内诗词，稳步提升语文成绩。

❤️ **关注我，每天解锁一首必考古诗词，陪孩子高效学国学、稳拿考试分！**
```

## 语言风格要点

### ✓ 要做的
- 用大白话讲解，孩子能听懂
- 用故事化语言描述诗人和背景
- 提供具体可用的答题模板
- 标注真正的易错点
- 给家长提供完整讲解话术

### ✗ 不要做的
- 不要堆砌辞藻、过度文艺
- 不要空谈意境、玄学解读
- 不要使用学术化表达
- 不要泛泛而谈考点
- 不要照搬文言直译

## 易错字标注示例

```
**易错提醒**：
- "霜"不要少写雨字头
- "望"不要写成"忘"
- "疑"右边是"矢"不是"失"
- 是小学默写高频错题
```

## 赏析语言示例

❌ **学术化表达**：
"诗人运用了比兴手法，借助自然景物的描摹，营造出一种清冷孤寂的意境氛围，进而抒发内心深处的羁旅之愁与故园之思。"

✓ **通俗化表达**：
"诗人把月光比作寒霜，写出了月光又白又冷的样子，也让人感受到深夜一个人在外的孤单，自然而然地想起了远方的家乡。"

## 考点标注示例

❌ **泛泛而谈**：
"本诗经常出现在考试中，学生要注意理解诗歌的思想感情。"

✓ **具体可落地**：
- 默写必考：全诗四句必须零失误默写
- 情境默写：常考"描写诗人望月思乡的诗句"，答"举头望明月，低头思故乡"
- 简答题：这首诗表达了诗人怎样的情感？答题模板：表达了诗人深夜望月、思念故乡的真挚情感
- 对比题：常与《泊船瓜洲》《九月九日忆山东兄弟》等思乡诗对比考察
