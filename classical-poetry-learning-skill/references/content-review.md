# 古诗词内容审校

按风险从高到低执行。审校不是“凭记忆再读一遍”，而是区分证据等级并暴露不确定性。

## 1. 事实层

- 逐字核对标题、原文、作者、朝代和标点。
- 核对生卒年、籍贯、称号与关键生平；享年和写作年龄不确定时用“约”。
- 创作年份、地点、动机和背景必须有依据。存在争议时写“约”“据主流说法”“一说”，不要编故事填空。
- 地名、典故、年号和历史事件不能张冠李戴。

## 2. 文学判断

- 依据句数、字数和格律常识判断体裁，不凭标题猜。
- 修辞、炼字、意象、情感和主旨都应能指回具体诗句。
- 区分作者明确表达、主流阐释和教学化延伸。
- 考点应像真实课堂/考试问题；删除空泛“万能模板”。

## 3. 分龄表达

- 检查句长、术语数量、抽象程度是否匹配目标年龄。
- 每个专业术语首次出现时用一句白话解释。
- 易错字提醒要具体且真实，不为凑模块标注普通字。
- 亲子话术应能直接说出口，避免论文腔和过度煽情。

## 4. 双语检查

- 中英文信息一一对应，不增删事实。
- 生平叙述通常用过去时；作品内容和分析通常用现在时。
- 检查主谓一致、冠词、单复数、介词、拼写和半角标点。
- 避免逐字中式英语；不确定时使用更短、更可靠的句子。
- 英译诗若不是有出处的经典译本，不要冒充权威译本，可标为“参考译文”。

## 5. 格式与产物

- 标题层级连续，诗句换行保留，重点标注适量。
- 七个必需内容齐全；可选扩展不超过两个且确有价值。
- 文件名、语言版本与正文一致。
- 文末话术不得包含用户未授权的账号、品牌或宣传承诺。

## 审校输出

若用户只要求审校，按以下格式报告：

1. **必须修正**：事实、原文、文学判断错误，附位置和理由。
2. **建议优化**：分龄、表达、结构和考点问题。
3. **未能核实**：需来源支持的细节及建议降级写法。
4. **最小修订方案**：按优先级给出可直接替换的文本。

若用户要求直接优化，修改后仍报告以上四项的摘要，并说明实际验证范围。
