# 古诗词公众号 HTML 主题库

把古诗词推文转成公众号可粘贴 HTML 时，从下面 11 个主题里选一个最贴合场景气质的。**不要固定用一个**——根据诗词情绪、读者对象（亲子/备考/文艺）、明暗偏好和装饰强度来选。

选定主题后，只加载对应的一个 `theme-*.md` 文件，用它的页面外壳 + 组件片段组织正文；每个主题在 `examples/` 下有可直接打开的 HTML 预览。

## 选择规则

1. **用户指定优先**：用户点名主题、说了明暗偏好或参考风格，就按用户来。
2. **没指定时按气质选**：
   - 追求最正统古典味 → **古典水墨**（默认首选）
   - 文艺慢读、意境优先 → 极简墨色
   - 诗人故事、背景丰富 → 复古杂志
   - 亲子启蒙、家长易读 → 暖白秩序 / 萌绿 / 手账便签
   - 备考默写、考点清单 → 青格笔记
   - 按诗词季节/色调呼应 → 秋暖用山吹、水景用雁栖湖、春生用萌绿、柔美用萌粉
3. **诗词情绪匹配画面**：豪放壮阔配深色重标题（古典水墨/复古杂志），婉约清新配浅色柔调（萌绿/雁栖湖/极简墨色）。
4. **和配图/封面风格协调**：如果推文已生成水彩暖色配图，优先选暖色系主题（古典水墨/山吹/暖白秩序），整篇视觉统一。
5. **可换不换死**：同一首诗可以换不同主题。没有强偏好时，先推荐一个并说明理由（基于视觉气质，不是题材硬绑定），让用户可换。

## 应用要点

- 用选定主题的页面外壳，再用它的组件片段组织正文；语义结构保持简单，样式全部内联。
- 不要混用不同主题的组件。某个组件不合适时，用该主题的设计令牌（颜色/间距）自行微调。
- 诗词原文用主题里最醒目的强调组件呈现（引用块/卡片/居中块），突出诗句本身。
- 发布稿正文不放封面副本、不放主标题 `<h1>`；图片用 HTTPS 地址，本地路径只用于预览稿。
- 正文不用 `<a>` 超链接；外部资料统一收到文末一个"参考资料"小节。
